LARLIB.jl

LARLIB.jl is a Julia library to perform geometrical computations on cellular complexes expressed through the Linear Algebraic Representation (LAR). This library is developed and maintained by the Computational Visual Design Laboratory (CVDLAB) of Università degli Studi di Roma Tre.

Dependencies

LARLIB.jl has several Julia dependencies:

  • NearestNeighbors

  • DataStructures

  • IntervalTrees

  • Triangle

Docstrings conventions

Bold is used to point out theory concepts. For example, look at the "1-skeletons" word in the docstring of LARLIB.skel_merge:

LARLIB.skel_merge — Method.
skel_merge(V1::Points, EV1::ChainOp, V2::Points, EV2::ChainOp)

Merge two 1-skeletons

source

Monospace is used for everything code related. Look this time at "container", "contained" and "Points" in the docstring of LARLIB.bbox_contains:

LARLIB.bbox_contains — Function.
bbox_contains(container, contained)

Check if the axis aligned bounding box container contains contained.

Each input box must be passed as the couple of Points standing on the opposite corners of the box.

source
