Official Julia implementation of [LAR](http://dx.doi.org/10.1016/j.cad.2013.08.044), the Linear Algebraic Representation for Solid Modeling.

## Installing

Just run `Pkg.add("LinearAlgebraicRepresentation.jl")`.
